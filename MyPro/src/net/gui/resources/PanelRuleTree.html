This panel displays a tree-structured graphical representation
of a parser-rule. It also supports context-menus for editing the rule-tree.
The context-menus are activated by right-clicking any rule-tree node.
Example rule-trees can be seen in the embedded sample grammars which can be
loaded by selecting "Help" -> "Sample grammars" -> ... from the main menu
